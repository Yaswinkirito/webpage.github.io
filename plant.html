DNA based storage have been in frontier of the 21st century as a solution to the increasing need for a data storage solution 1 gram of DNA can store about an exabyte (1 billion GB) of data. Two undergraduate students successfully encoded a python program into the DNA of a plant and it’s was decoded from it’s seed successfully. Many companies have emerged in the recent years based on the idea of plant-based cloud storage. This method provides a viable alternative to the current method where data are stored in server farms which run on electricity made by fossil fuels. The above method is not so difficult to understand since it is a simple conversion of machine code(binary language) to the four letter(A,T,G,C) code and it’s a medium which require a low maintenance.
